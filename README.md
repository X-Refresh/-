# Сайт ежегодной конференции Института НМСТ

## О проекте
Данный репозиторий содержит исходный код клиентской части (frontend) веб-сайта для ежегодной научно-практической конференции Института НМСТ. Проект разрабатывается в рамках курсовой работы по созданию ИТ-инфраструктуры для образовательного учреждения.

Сайт предназначен для информирования участников о датах проведения, программе мероприятия, а также служит точкой входа для регистрации, оплаты оргвзноса и подачи научных тезисов.

## Технологии
* **HTML5** — семантическая разметка структуры страниц.
* **CSS3** — стилизация интерфейса, использование Flexbox для адаптивной верстки и CSS-переменных для поддержания строгой цветовой схемы (темная тема с корпоративными зелеными акцентами).

## Структура проекта
* `index.html` — главная страница сайта (Hero-блок, навигация, дедлайны).
* `css/style.css` — основные каскадные таблицы стилей.
* `img/` — директория для хранения графических материалов (логотипы, фоновые изображения).

## Ключевой функционал
* Адаптивная навигационная панель.
* Информационный блок с ключевыми датами (период Early Bird, дедлайн загрузки тезисов).
* Пользовательский интерфейс (UI), спроектированный с учетом специфики научной среды.
